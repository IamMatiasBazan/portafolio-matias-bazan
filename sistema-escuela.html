Sistema Web Educativo: Proceso de Diseño y Desarrollo Completo​



Para crear este sistema para una escuela, lo desarrollé utilizando HTML, Bootstrap, PHP, y un poco de CSS y JavaScript. La base de datos utilizada fue MySQL, gestionada a través de phpMyAdmin. Para comenzar, primero diseñé la lógica de mi aplicación: elaboré un diagrama entidad-relación y un diagrama de flujo. Utilicé la herramienta Lucidchart para llevar a cabo este proceso de diseño lógico.




Luego, pasé a prototipar el diseño en Figma, definiendo cómo se vería mi aplicación a nivel frontend. Tal como describí en el proceso de mi portafolio, trabajé con un ancho de 1920 px para monitores Full HD, estableciendo un ancho de contenido de 1140 px. Existen monitores más grandes, pero diseñar una página web pensando únicamente en el 1 % o 2 % de la población no sería lógico. Aunque seguí principios fundamentales de diseño como alineamiento, repetitividad y proximidad, al principio no tuve en cuenta adecuadamente el ancho de 1140 px para el contenido. Como ya estaba trabajando con Bootstrap, también aproveché su librería de iconos. Bootstrap es un framework frontend de código abierto que proporciona recursos en HTML, CSS y JavaScript, ideal para el desarrollo de aplicaciones web.
